Welcome to our Blockchain Program! Here are the instructions and key features of the program:

1. Getting Started

When you first start the program, the system will check the database. If no blocks exist, the Genesis Block (the first block) will be created automatically.

รูปที่ 1: แสดงถึงการ โอนคะแนนที่ได้จากเกมส์ นำไปเก็บเป็นเหรียญลงในกระเป๋า

รูปที่ 2: เมื่อโอนสำเร็จ จะขึ้นลักษณะดังกล่าว

รูปที่ 3: การบันทึกข้อความเพื่อ นำไปบรรจุลงบล็อกเชน

รูปที่ 4: เมื่อกด Save Message จะขึ้นหน้าต่าง แจ้งว่า ระบบจะโอนเหรียญที่ได้จากการปิด บล็อก ลงในกระเป๋าใบแรกที่ระบบสร้างขึ้นใหม่

รูปที่ 5: แสดงหน้า กระเป๋าที่สามารถสร้างใหม่ได้ไม่จำกัด

รูปที่ 6: การโอนเหรียญไปกระเป๋าอื่นๆ

รูปที่ 7: แสดงธุรกรรมของการโอนเหรียญ

รูปที่ 8: การเก็บ ฐานข้อมูลบล็อก เข้าสู่ Server หลัก เพื่อเก็บรักษาข้อมูล และมีการใช้งานในอนาคต

รูปที่ 9: การโอนฐานข้อมูลต้อง สมัครเพื่อใช้ ไอดี ในการเก็บชื่อฐานข้อมูล

รูปที่ 10: หน้าเข้าสู่ ระบบ ก่อนจะทำการโอนฐานข้อมูลเข้าสู่ระบบ

รูปที่ 11: การเลือกฐานข้อมูลที่สร้าง จะอยู่ตำแหน่งเดียวกับที่ รันโปรแกรม ** ต้องโอน ไฟล์ ที่ชื่อว่า blockchain.db เท่านั้น

รูปที่ 12: ไฟล์ที่ถูกบันทึกสมบูรณ์ จะขึ้นในรายการฐานข้อมูลหลัก ตาม ไอดีของผู้ที่โอน

2. Storing Messages in the Blockchain

Users can input a new message, and the program will create a new block with the necessary details such as Index, Message, Timestamp, Previous Hash, and Current Hash.
